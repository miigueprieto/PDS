# Edición de imagenes

En este proyecto se programarán una serie de filtros que se aplicarán a cualquier tipo de imagen para obtener una versión procesada de la imagen original, con fines meramente estéticos o más técnicos, como la mayor facilidad de detección de algunos elementos de la imagen.

Filtros desarrollados:
- Filtro Homomórfico
- Filtro Mediana
- Filtro Notch
- Filtro Convolución
- ...
